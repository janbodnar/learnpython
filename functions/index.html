Funkcie

Funkcia je blok znovupoužiteľného kódu v Pythone. Úlohou funkcie je vykonať nejakú špecifickú úlohu. Funkcie redukujú duplicitu kódu, zlepšujú čitateľnosť zdrojového kódu, umožňujú nám rozdeliť zložité úlohy do jednoduchších podúloh.

Funkcie v Pythone sú first-class citizens. To znamená, že ich môžeme priradiť premenným, uložiť do kolekcií, a predať ako argument iným funkciám. Vďaka tomu môžeme v Pythone vykonávať niektoré pokročilé programovacie techniky.

Typy funkcií v Pythone

Existujú dva druhy funkcií: zabudované a uživateľsky definované. Zabudované funkcie sú súčasťou jazyka Python. Napríklad funkcie dir(), len(), alebo abs(). Uživateľsky definované funkcie sa vytvárajú pomocou kľúčového slova def.

Zabudované funkcie rozdeľujeme na built-in funkcie a funkcie dostupné v externých moduloch.

functions.py
#!/usr/bin/python3

from math import sqrt

def cube(x):
    return x * x * x

print(abs(-1))
print(cube(9))
print(sqrt(81))

Funkcia cube() je uživateľsky definovanou funkciou. Funkcia abs() je built-in funkciou. A funkcia sqrt() je funkcia dostupná v externom module (math).

functions2.py
#!/usr/bin/env python3

n = (1, 2, 3, 4, 5)

def stats(x):

    mx = max(x)
    mn = min(x)
    ln = len(x)
    sm = sum(x)

    return mx, mn, ln, sm

mx, mn, ln, sm = stats(n)

print(mx, mn, ln, sm)

V príklade vidíme využitie zabudovaných funkcií max(), min(), len() a sum().

Argumenty funkcií

Funckie môžu (ale nemusia) prijímať argumenty. Argumenty sú hodnoty, ktoré sa posielajú funkciám. Funckie tieto argumenty spracujú a voliteľne vrátia nejakú hodnotu späť pomocou kľúčového slova return.

fahrenheit.py
#!/usr/bin/env python3

# fahrenheit.py

def C2F(c):
    return c * 9/5 + 32

c1 = C2F(100)
c2 = C2F(0)
c3 = C2F(30)

print(c1, c2, c3)

V príklade máme uživateľsky definovanú funkciu C2F(), ktorá prepočíta teplotu v stupňoch Celzia na Fahrenheit.

Funkcie môžu mať tiež argumenty s implicitnými hodnotami. Tie sa využijú, ak nie sú funkciám zadané hodtnoty.

implicit_args.py
def power(x, y=2):

    r = 1

    for i in range(y):

        r = r * x

        return r

print(power(3))
print(power(3, 3))
print(power(5, 5))

V príklade máme funkciu power(), ktorá má dva argumenty. Druhý argument má implicitnú hodnotu 2.

Knižnica NumPy

NumPy je fundamentálny balík pre vedeckú (matematickú) prácu v Pythone. Na tejto knižnici sú ďalej postavené ďalšie dôležité a populárne knižnice (napr. Pandas, Matplotlib), ktoré sa využívajú v data science a machine learning.

matrixes.py
#!/usr/bin/env python3

import numpy as np

a = np.array([1, 2, 3])
print(a)
print("********************")

b = np.array([[1, 2, 3], [4, 5, 6]])

print(b)
print("********************")

print(2*b)
print("********************")

print(a*b)
print("********************")

Tento príklad pracuje s maticami.

Tento príklad pracuje využíva trigonometrické funkcie.

line_chart.py
#!/usr/bin/python3

import numpy as np
import matplotlib.pyplot as plt

t = np.arange(0.0, 3.0, 0.01)
s = np.sin(2.5 * np.pi * t)

plt.plot(t, s)
    
plt.xlabel('time (s)')
plt.ylabel('voltage (mV)')

plt.title('Sine Wave')
plt.grid(True)

plt.show()

Tento príklad využíva NumPy pre vytvorenie dátových radov. Tie sa použijú pri tvorbe čiarového grafu.

Syntax jazyka Python

V tejto časti si povieme niečo o základoch syntaxe Jazyka Python.

Podobne ako ľudské jazyky, majú aj programovacie jazyky svoju syntax. Syntax jazyka Python je množina pravidiel ako písať Python program a ako bude program interpretovaný (behovým systémom a programátormi).

Program v jazyku Python sa skladá z atomických prvkov, ktoré nazývame tokeny. Základné tokeny v Pythone sú:

  • Komentáre
  • Premenné
  • Literály
  • Operátory
  • Odsadenie
  • Oddeľovače
  • Kľúčové slová

Komentáre

Komentáre slúžia na objasnenie zdrojového kódu. Komenty v jazyku Python sú uvodené znakom #.

comments.py
#!/usr/bin/python3

# comments.py
# Author Jan Bodnar
# ZetCode 2018

def main():
    print("Comments example")

main()

Všetko, čo nasleduje po znaku # Python interpreter ignoruje.

Premenné

Premenná je identifikátor, ktorý nesie/obsahuje hodnotu. V programovaní hovoríme, že premennej priraďujeme hodnoty. Z techického hľadiska je premenná referenciou v pamäti počítača, kde je hodnota uložená. V jazyku Python môže premenná obsahovať reťazce, čísla, rôzne objekty, a funkcie. Počas behu programu môžme premennej priradiť rôzne hodnoty.

Premenné v Pythone sa môžu skladať z alfanumerických znakov a podčiarkovníka. (underscore). Názvy premenných nesmú začínať číslami. 

Value
value2
company_name

Toto sú príklady platných identifikátorov. 

12Val
exx$
first-name

Toto sú príklady neplatných identifikátorov.

Názvy premenných v Pythone rozlišujú veľkosť písmen. Preto Price, price, a PRICE sú tri rôzne premenné. 

#!/usr/bin/python3

number = 10
Number = 11
NUMBER = 12

print(number, Number, NUMBER)

Aj keď môžme definovať tri rôzne druhy premenných veľkosťou písmen, rozhodne sa to neodporúča.

Literály

Literál je notácia, ktorá reprezentuje hodnotu v zdrojovom kóde. 

age = 29
nationality = "Hungarian"

V tomto príklade priradíme dve hodnoty premenným age a nationality. Číslo 29 a režazec "Hungarian" sú literálmi.

Operárory

Operátor je symbol, ktorý sa používa na vykonanie akcie na hodnotách. 

+    -    ~    *    **    /    //
%    <:<:    >>    &    |    ^
and    or    not    in    not in 
is    is not    <:    >    !=    <:>
==    <:=    >=

Toto je zoznam operátorov v Pythone.

operators.py
#!/usr/bin/env python3

x = 23
y = 43

z = x * y / 10

print(z)

age = 19
 
if (age > 18):
    print("eligible for driving licence")

nums = (2, 4, 6, 8, 10)
val = 6

if val in nums:
    print("value is in the nums tuple")
else:
    print("value is not in the nums tuple")

Príklad ukazuje použitie niekoľkých operátorov.

Odsadenie

Odsadenie sa používa v Pythone na určenie hraníc blokov kódu. Väčšina programovacích jazykov (napr. Java, C, PHP) používa na tvorbu blokov znaky zložených zátvoriek {}. Na odsadenie sa odporúča použiť štyri medzery. 

if age < 18
    print("adult person")

for i in range(5):
    print(i)

Za kľúčovým slovom if sa očakáva blok. Nový príkaz začína na novom riadku, odsadený čtyroma medzerami. Cyklus for začína nový blok kódu.

Oddeľovače

Oddeľovače (delimiters) sú špeciálne znaky, ktoré určujú hranice medzi separátnymi úsekmi kódu. 

(       )       [       ]       {       }      
,       :       .       `       =       ;
+=      -=      *=      /=      //=     %=
<:=      |=      ^=      >:>:=     <:<:=     **=
'       "       \       @

Oddeľovače sa využívajú na tvorbu výrazou, reťazových literálov, ntíc, slovníkov, alebo zoznamov.

operators.py
#!/usr/bin/env python3

nums = (1, 2, 4, 5, 6)
print(nums)

name = "Jack Sparrow"
print(name)

print(name.upper())

Tento príklad ukazuje využitie niekoľkých oddeľovačov.

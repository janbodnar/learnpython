Inštalačný manažér pip

Python pip je oficiálny inštalačný manažér jazyka Python. Slúži na inštaláciu externých distribučných balíkov. Plní podobné úlohy ako inštalačný manažér composer v jazyku PHP alebo npm jazyka JavaScript.

Pip inštaluje balíček a zároveň všetky jeho závisloti, čím výrazne znednodušuje prácu programátorov. Každá verzia Pythonu má svoj vlastný pip manažér.

Repozitár PyPi

Distribučné balíky programovacích jazykov sú uskladnené v repozitároch, z kadiaľ si ich môžeme stiahnuť a naištalovať. Oficiálnym repozitárom jazyka Python je PyPi: pypi.org.

Anaconda

Anaconda je inštalačný manažér a manažér virtuálnych prostredí vyvinutý matematikmi pre data science a machine learning. Umožňuje inštaláciu distribučných balíkov jazykov Python, R, Fortran a ďalších. Anaconda a pip sú oddelené systémy.

Pip príkazy

Na inštaláciu distribučného balíčka použijeme príkaz pip install. 

$ pip install humps

Tu si nainštalujeme balík humps, ktorý konvertuje reťazec na formát camel case.

camel_case.py
#!/usr/bin/env python3

from humps import camel

s1 = "thick forest"

print(camel.case(s1))

s2 = "FORTRAN LANGUAGE"

print(camel.case(s2))

Príklad transformuje dva reťazce na formát camel case.

Na odinštalovanie balíka použijeme príkaz pip uninstall. 

$ pip uninstall humps

Zmažeme balík humps. Ďalej ho už nemôžeme používať. 

$ python camel_case.py
thickForest
fortranLanguage

Dostaneme takýto výstup. 

$ python camel_case.py
Traceback (most recent call last):
  File "camel_case.py", line 3, in <module>
    from humps import camel
ModuleNotFoundError: No module named 'humps'

Po tom, čo sme odinštalovali balík, už program nevieme spustiť. Python nám zahlási chybu, že nepozná modul humps.
